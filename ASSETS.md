## 7. CHROME DESIGN TOKENS (from calibre-os/app/globals.css)

These are **ready to copy** into Requite's `app/globals.css` in Stage 1:

```css
/* Colour palette */
--color-navy: #0D1B35
--color-teal: #00C4A0        /* primary chrome accent */
--color-teal-light: #00E5BC
--color-iris-indigo: #6366F1  /* chrome spectrum 1 */
--color-iris-violet: #A855F7  /* chrome spectrum 2 */
--color-iris-rose: #EC4899    /* chrome spectrum 3 */

/* Animations */
@keyframes aurora-drift   /* 18s, background glow */
@keyframes chrome-shift   /* 7s, text/border shimmer */

/* Classes */
.aurora-bg        /* ambient glow background */
.chrome-text      /* rainbow gradient text (for headlines, logo) */
.iris-divider     /* 2px rainbow horizontal rule */
.iris-border      /* gradient-border card wrapper */
.btn-iris-sheen   /* hover sheen effect on buttons */
.iris-progress    /* rainbow progress bars */
```

Font: `Plus Jakarta Sans` (display/body) + `JetBrains Mono` (mono).
